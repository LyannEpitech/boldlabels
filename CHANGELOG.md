# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-13

### Added
- Initial release of BoldLabels
- Template editor with React-Konva canvas
- Support for text, barcode, QR code, image, and rectangle elements
- CSV import with column mapping
- PDF generation with customizable layout
- Preset templates (Avery 5160, 5163, 5164, 5167)
- Docker deployment support
- Backend API with Express and Prisma

### Features
- Drag & drop element positioning
- Grid snapping and alignment tools
- Undo/redo functionality
- Layer ordering
- Real-time PDF preview
- Responsive design

### Technical
- React 18 + TypeScript frontend
- Node.js 20 + Express backend
- PostgreSQL database
- Zustand state management
- jsPDF for PDF generation
- Docker Compose deployment
